## 17개 제작 Case 공정 흐름

| Case | 상품유형 | 공정 흐름 |
|------|--------|---------|
| 1 | 낱장인쇄 | 디지털출력 → 코팅 → 재단 → 1차포장 |
| 2 | 스티커 | 디지털출력 → 코팅 → 반칼/완칼커팅 → 재단 → 1차포장 |
| 3 | 인쇄후가공 | 디지털출력 → 코팅 → 재단 → 오시/접지/타공/귀돌이/수동도무송 → 1차포장 |
| 4 | 책자(링제본) | 디지털출력 → 코팅 → 재단 → 링제본 → (재단) |
| 5 | 책자 | 디지털출력 → 코팅 → 재단 → 중철제본/PUR제본/떡제본 → 재단 → 1차포장 |
| 6 | 커버/클립보드 (*레더+싸바리) | 디지털출력 + 패브릭출력 → 싸바리 → 조립/후가공 → 1차포장 |
| 7 | 실사/사인 | 실사출력 → 쿨코팅 → 조립/후가공 → 1차포장+박스포장 |
| 8 | 투명실사 | 라텍스(화이트)출력 → 조립/후가공 → 1차포장+박스포장 |
| 9 | 커팅 | 레이저커팅/시트커팅 → 1차포장+박스포장 |
| 10 | 봉제미싱 | 패브릭출력 → 봉제미싱 → 1차포장 |
| 11 | UV단품 | UV평판출력 → 1차포장 |
| 12 | 투명클립보드 | UV평판출력 → 레이저커팅 → 조립/후가공 → 1차포장 |
| 13 | 아크릴가공 | UV평판출력 → 레이저커팅 → 아크릴가공 → 1차포장 |
| 14 | 전사 | 전사인쇄 → 1차포장 |
| 15 | 도장 | 도장제작 → 1차포장 |
| 16 | 외주공정 | 동판제작 → 박작업 → 생산본부 |
| 17 | 재고상품 | 상품액세서리 → 1차포장 |
| 18 | 외주제작 | 책자제작/실사현수막/합판인쇄/고주파 → 1차포장 |

## 공정 상태 트래킹 포인트

| 부서 | 트래킹 포인트 |
|------|------------|
| 디지털출력팀 | 출력완료 |
| 스티커가공파트 | 커팅완료 |
| 인쇄후가공팀 (*제본상품) | 제본시작 |
| 특수인쇄가공팀 | - |
| 봉제가공파트 | 봉제시작 |
| 아크릴가공 | 가공시작 |
| 1차포장 | 제작완료 |
| 박스포장 | 송장출력 |

## 입고 트래킹
봉제미싱입고 / 아크릴가공입고 / 제본입고 / 박작업입고
**규칙**: 앞공정은 안찍으면 다음공정 못찍히게 (순차적 공정 관리)

- 스티커 일별 출고건(량) / 미출고건(량)
- 봉제미싱 입고건(량) / 미출고건(량)
- 아크릴가공 입고건(량) / 미출고건(량)
- 제본입고건(량) / 미출고건(량)

- 공정별 평균리드타임
- 공정별 평균 제작기간
